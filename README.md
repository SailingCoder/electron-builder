# electron-builder

## To Use

```bash
# Clone this repository
git clone https://github.com/ghSailing/electron-builder.git
# Go into the repository
cd electron-builder
# Install dependencies
npm install
# Run the app
npm start
```

### make

```bash
# make out
npm run make
```
### build windows exe

```bash
# build exe
npm run package
```

### build mac dmg

```bash
# build dmg
npm run dmg
```

> mac打包dmg，必须在mac电脑
